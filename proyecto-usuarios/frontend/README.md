# SANMIT - Aplicación Usuarios (React + Node + MySQL)

**Integrantes:** Miguel Martinez y Sebastian Montiel

## Descripción
Aplicación que permite CRUD de usuarios y login. Diseño con identidad SANMIT.

## Cómo ejecutar
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start

## Ramas
main, develop, feature/crud, feature/login, feature/diseno-sanmit

## Endpoints principales
GET /api/usuarios
POST /api/usuarios
PUT /api/usuarios/:id
DELETE /api/usuarios/:id
POST /api/usuarios/login
